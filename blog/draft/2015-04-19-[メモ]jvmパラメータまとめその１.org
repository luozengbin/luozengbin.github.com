* PrintGCCause
GCを起こしたイベント情報を出力する

* CMS GCパラメータ

|------------------------------------+------------------------------------------------+--------------|
| 項目名                             | 説明                                           | デフォルト値 |
|------------------------------------+------------------------------------------------+--------------|
| UseCMSInitiatingOccupancyOnly      |                                                |              |
| CMSScavengeBeforeRemark            | CMS実施前に強制的MinorGCを行う                 |              |
| CMSClassUnloadingEnabled           | Perm領域の掃除もCMS方式で行う                  |              |
| CMSClassUnloadingMaxInterval       | TODO: 調査                                     |              |
| CMSInitiatingPermOccupancyFraction | CMSの起動条件を示す、Perm領域の使用しきい値(%) |              |
|------------------------------------+------------------------------------------------+--------------|

* System.gc() に関わるパラメータ

|----------------------------------------------+-----------------------------------------------+--------------|
| 項目名                                       | 説明                                          | デフォルト値 |
|----------------------------------------------+-----------------------------------------------+--------------|
| DisableExplicitGC                            | 強制的GCを無視する                            | false        |
| ExplicitGCInvokesConcurrent                  | 強制的GC時にCMS方式で行う（ヒープのみ）       | false        |
| ExplicitGCInvokesConcurrentAndUnloadsClasses | 強制的GC時にCMS方式で行う（ヒープとPerm領域） | false        |
|----------------------------------------------+-----------------------------------------------+--------------|
